## ✨ Features

### 🌟 Client-Facing Features
* **Dynamic Discovery:** Browse salons, explicit service listings, and curated packages loaded directly from the database.
* **Smart Appointment Booking:** Seamless scheduling wizard to pick dates, times, and specific salon services.
* **Robust User Profiles:** Complete onboarding with Email or Phone Number signup/login. Users can update avatars, manage contact details, and view history.
* **Appointment Control Center:** A dedicated dashboard for clients to track, reschedule, or cancel upcoming appointments in real time.

### 🎨 UI/UX Highlights
* **Cinematic Motion:** Smooth, immersive scroll-triggered animations and micro-interactions powered by GSAP.
* **Premium Aesthetic:** A clean, minimalist layout with a dark/light balanced theme tailored for the beauty and wellness industry.
* **Fully Responsive:** Fluid layouts designed to work perfectly on mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack

### Frontend
* **HTML5 & CSS3:** Semantic markup and modern layout architectures (Flexbox/Grid) with custom properties.
* **Vanilla JavaScript (ES6+):** Clean, modular script architecture for state handling, DOM manipulation, and API consumption.
* **GSAP (GreenSock Animation Platform):** High-performance UI animations and smooth-scrolling experiences.

### Backend Capabilities (Data-Driven)
* **Dynamic API Integration:** All salon data—including names, descriptive bio sections, pricing, and package bundles—is fetched asynchronously from the database layer.
* **Secure Authentication:** Secure user verification endpoints handling phone/email login and profile persistence.
